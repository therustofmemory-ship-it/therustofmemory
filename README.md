# TRM — The Rust of Memory Website

A responsive service website for **TRM — The Rust of Memory**, presenting digital freelance services and local mobile repair support.

## Features

- Fully responsive layout for desktop, tablet and mobile
- Premium light/dark mode interface
- Mobile-friendly navigation menu
- Digital services page
- Mobile repair services page
- WhatsApp and email contact buttons
- Smooth animations with reduced-motion support
- GitHub Pages-ready static files

## Pages

- `index.html` — Main service hub
- `freelance.html` — Digital / freelance services
- `hardware.html` — Mobile repair services

## Deployment

Upload all files and folders to the root of your GitHub repository, then enable GitHub Pages from:

`Settings → Pages → Deploy from a branch → main → /root`
